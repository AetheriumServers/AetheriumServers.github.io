TTT Rules!

READ THEM! There's important info here!

Innocents/Detectives

  1. RDM (Random Death Match) is strictly prohibited
  2. T-baiting(Random shoooting/saying you're the Traitor) is KoSable
  3. False KoS will result in a slay
  4. Use of the Golden Deagle must come with reasonable suspicion. If by chance, you kill a T with it, it will still count as RDM
  5. Player's may always defend themselves against attacks or harm from other players.(applies to only the person attacked)
  6. Suspicion killings are prohibited. If you think someone might be the traitor, you must first find concrete evidence before killing them.
  7. Killing terrorists who are following you is prohibited
  8. Killing AFK's (Away From Keyboard) is prohibited. This server will automatically slay the afk's after some time.
  9. Prop killing is considered RDM unless used by a traitor.
  10. Rooms can be claimed only if the round is in haste mode.

Traitors

  1. Killing fellow traitors is prohibited
  2. Try not to draw out the round

Reporting Rulebreakers/RDMers

  • Type !report for Rdmers.
  • If no admin is available, note the players name and their actions, then let an admin know as soon as they are available, Or post it on the Steam group under TTT Player Reports.
